# 一分钟 · Feel One Minute

> 一个 24 小时 GameJam 作品,主题 **「1min(一分钟)」**。
> 不靠读秒,而是逼你用**身体感觉**去估测「一分钟到底有多长」。

按住开始、松手结束,目标是在**第 60 秒**松手——**但不能超过**,超过就「爆掉」。
整个过程**不会显示任何走动的时间**,全凭体感。

---

## 🎮 玩法

1. 选择人数(**1–4 人**,本地同屏轮流 / 同时玩)。
2. 每个玩家**按住一个键**(键盘),或**按住屏幕上的方块**(触摸)加入。
3. 凑齐人数 → **3、2、1 开始** → 进入计时。
4. 凭感觉在第 60 秒**松手**:
   - 没超过 60 秒、且**最接近**的人**获胜**;
   - **超过 60 秒直接爆掉**判负。
5. 结算看排名,可「再来一局」或「换人数」。

> ⚠️ **核心设计**:计时过程中绝不显示秒数、进度条或任何走动的时间——这是游戏的灵魂。
> 结果只在松手后揭晓:实际时长、与 60 秒的差值、名次。

## ✨ 特性

- **本地多人(pass-and-play)**:1–4 人,各自按住一个键 / 屏幕方块。
- **手机触摸支持**:屏幕方块即触点,移动端可玩。
- 开局 3-2-1 倒计时横幅、到 60 秒自动爆掉、结算排名。
- 失焦 / 指针丢失兜底,计时不卡死。

## 🛠 技术

- **单文件**:全部在一个 [`index.html`](index.html) 里(HTML + CSS + JS)。
- **纯原生**:vanilla JavaScript,无框架、无依赖、**零构建**。
- **计时用时间戳差值**(`performance.now()`),精度优先,不逐帧累加。

## 🚀 运行

直接**双击 `index.html`** 用浏览器打开即可——不需要安装、不需要构建。

> 本版本无外部图片/音频资源,`file://` 直接打开就能跑。
> 若以后加入本地音频/图片,本地自测请改用 `python -m http.server` 起个服务器。
